# 100 Days of Python — Robotics & AI

One project a day for 100 days. Difficulty scales with the day number.
Building toward autonomous robot simulation, computer vision, and deep learning.

**Progress:** 1 / 100

---

## Rules

- **Days 1–40:** no AI-generated code. Docs first; AI only explains concepts after 30+ min stuck.
- **Days 41–70:** AI may explain errors, never write solutions.
- **Days 71–100:** AI for boilerplate only. All core logic is mine.
- Never run a line I can't explain out loud.
- Ship the ugly version. Cut scope, don't skip days.
- One commit per day.
